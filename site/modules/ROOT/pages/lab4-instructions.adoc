= Lab 4 Instructions: MCP Retrieval

Follow these notebooks to search a Neo4j knowledge graph through the Model Context Protocol, progressing from vector search to graph-enriched retrieval to hybrid search.

TIP: For concepts and background, see xref:lab4.adoc[Lab 4: MCP Retrieval].

== Prerequisites

* Completed **Lab 1** (Neo4j Aura instance created and running)
* Data loaded into Neo4j with embeddings, entity nodes, and fulltext indexes (from the data loading pipeline)
* `CONFIG.txt` at the project root filled in with Neo4j credentials, `MCP_GATEWAY_URL`, and `MCP_ACCESS_TOKEN`
* A running environment (SageMaker Studio or GitHub Codespace)

== Notebooks

[cols="1,2,3"]
|===
| Notebook | Title | What You Build

| `01_vector_search_mcp.ipynb`
| Vector Search via MCP
| Semantic vector search with Bedrock Nova embeddings through an MCP agent

| `02_graph_enriched_search_mcp.ipynb`
| Graph-Enriched Search via MCP
| Vector search enriched with document metadata, neighboring chunks, and entity traversal (companies, products, risk factors)

| `03_fulltext_hybrid_search_mcp.ipynb`
| Fulltext and Hybrid Search via MCP
| Keyword search with Lucene operators and agent-driven hybrid search combining vector and fulltext via custom `@tool` wrappers
|===
